# Elite Fashion Store

Elite Fashion Store is a Streamlit-based eCommerce web application for browsing clothing products, placing orders, and tracking purchases.

## Features

- Product catalog for clothing items
- User-friendly shopping interface
- Order placement
- Order tracking
- Cash on Delivery and online payment option display
- Simple data storage using JSON files
- Streamlit web app interface

## Project Files

- `app.py` – main Streamlit application
- `products.json` – product data
- `users.json` – user data
- `orders.json` – order data
- `requirements.txt` – Python dependencies
- `Procfile` – optional deployment file
